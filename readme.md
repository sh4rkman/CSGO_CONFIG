#### csgo-npp-language 

"CSGO ConfigFile" is a text highlight for Counter-Strike Global Offensive configuration files
- Copy the xml file I wrote
- Open Notepad++ and go to Language > Define your Language > Import
- Select the file
- Tadaaaaa magic happends
